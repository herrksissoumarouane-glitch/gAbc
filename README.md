# gAbc
Kurze Beschreibung:
Dieses Projekt ist ein Verwaltungssystem für Dozenten und Abwesenheiten an einer Schule.

Meine Arbeit / Implementierte Funktionen
1. Dozentenverwaltung

Import von Dozentendaten in die Datenbank basierend auf dem Diplomtyp

Anzeige der Dozenten nach Diplomtyp

Zuweisung jedes Dozenten zu den entsprechenden Gruppen

2. Abwesenheitsverwaltung

Hinzufügen von Abwesenheiten für ausgewählte Gruppen

Verwaltung der Abwesenheiten für jede Gruppe

3. Download-Funktion

Herunterladen der Abwesenheitsdaten nach Woche und Abteilung

Export der Daten in Dateien für die weitere Verwendung
